# Waste Logistics Engine

**A multi-tenant logistics and asset management platform for the waste industry.**

This project is a full-stack SaaS solution designed to handle inventory tracking, automated dispatching, and customer bookings for dumpster rental businesses.

This platform is built with a focus on high-concurrency performance and strict type safety.

- **Backend:** Go (Golang) - chosen for high-throughput API performance.
- **Frontend:** Next.js (TypeScript) - for a reactive, SEO-friendly user interface.
- **Database:** PostgreSQL - utilizing relational data modeling for complex order states.
- **Infrastructure:** Docker & Docker Compose.
